# OpenShift 简介

## 什么是 OpenShift
- 红帽
- 开源的容器应用 Paas 平台
- Kubernetes 和 docker
- 帮助开发人员快速在云端开发、托管、扩展、交付应用。

> 云计算的三种服务模式
> - Iaas 基础设施即服务
> - Paas 平台即服务
> - Saas 软件即服务

## OpenShift Online
- 分层系统
  - docker
  - k8s
  - 源代码管理、构建、部署
  - 大规模管理和发布镜像
  - 大规模管理应用
  - 用户管理
  - 网络基础设施
- 系统架构
  - 微服务架构
  - 使用restful API暴露核心对象
  - 使用Controller获取核心对象的状态，执行相关的操作，并修改对应的状态
